Lexical analyzer
The program works in two modules:
First module - print all tokens:

	Keywords: if, inline, int, long, register, restrict, return, short, signed, sizeof, static, struct, switch, typedef, union-yellow

	Identifiers-orange

	Floating-constants-red

	String-literals-blue

	operations(/, /=, %, %=, %>, %:, %:%:, <, <<, <<=, <=, <:, <%, >, >=, >>, >>=, =, ==, !=, |, |=, &&, &=, *=, ^, ^=, :>, +=, -=)-pink

	comments -green


Second module - print only string literals

To launch the program you should enter file and way of work: wright all tokens - "all" or only string literals - "string" in command line
